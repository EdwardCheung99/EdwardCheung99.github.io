WhatChat - An Online Chatroom Application

WhatChat is an application that allows users to chat in real time chat rooms with one another.

Users can both join chat rooms as well as create new chat rooms. Existing chat rooms contain the logs of messages which were previously sent in that room. Users must set their usernames in a chat before they can send messages. Users can also filter a specific chat room for messages based on the name of the user who sent it, and/or the content within the message.

WhatChat utilizes web sockets via Socket.IO to perform real time communications. The Front End was developed in HTML and CSS (Bootstrap) with Handlebars.js templating. The Back End was developed with Node.js and Express.js. PostgreSQL is used to store message and chat data.
